# Unity Scripting Playground

## Descripción

Este proyecto funciona como mi libreta de apuntes, entorno de experimentación y laboratorio de aprendizaje para el desarrollo con Unity utilizando C#.

Su propósito es servir como un espacio donde puedo explorar, probar y documentar conceptos fundamentales y avanzados relacionados con la programación dentro del motor Unity, permitiéndome comprender mejor su funcionamiento mediante ejemplos prácticos y pequeños experimentos aislados.

## Objetivos

* Aprender los fundamentos de C# aplicados a Unity.
* Comprender el ciclo de vida de los componentes y GameObjects.
* Experimentar con estructuras de datos y algoritmos.
* Explorar patrones de programación comunes en videojuegos.
* Documentar hallazgos, comportamientos y buenas prácticas.
* Construir una base sólida para proyectos futuros.
